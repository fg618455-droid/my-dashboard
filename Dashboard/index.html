iOS Shortcut — Apple Health Bridge

Dieser Shortcut läuft täglich automatisch auf deinem iPhone, liest Daten aus Apple Health (inkl. Cal Plus) und schickt sie direkt in deine Supabase-Datenbank.

Schritt 1 — Öffne die Kurzbefehle-App auf deinem iPhone
→ Neuen Kurzbefehl erstellen → „Aktion hinzufügen"
Schritt 2 — Folgende Aktionen in dieser Reihenfolge hinzufügen:
1. Gesundheit → Menge aufzeichnen abfragen:
   • Aktive Energie (kcal) → Variable „burned"
   • Schritte → Variable „steps"
   • HRV (ms) → Variable „hrv"
   • Im Bett verbrachte Zeit → Variable „sleep_h"
2. Cal Plus → Apple Health synchronisiert Kalorien automatisch als „Nahrungsenergie"
   → Ebenfalls abfragen → Variable „cal_in"
Schritt 3 — Aktion „URL abrufen" (POST-Request):
Methode: POST · Header: apikey + Authorization (dein Supabase Anon Key)
Body: JSON mit den Variablen
POST → https://rdkqtegwkzfdvjphfjfw.supabase.co/rest/v1/health_logs
Body: { "date_str": "DATUM", "calories_in": CAL_IN, "calories_burned": BURNED, "steps": STEPS, "sleep_hours": SLEEP_H, "hrv": HRV }
Schritt 4 — Automatisierung einrichten:
Kurzbefehle → Automatisierung → + → Tageszeit → 22:00 Uhr täglich
→ Deinen neuen Shortcut auswählen → „Ohne Rückfrage ausführen"
